W3 Selectors

  • Syntax: w3.action(selector)
  • The action() is performed on the selected element(s).
  • The (selector) selects the HTML element(s).
  • You can use CSS selectors to target elements.
  • ( ) In red shows how the element can be targeted.
  • To see change click Show/ Hide /Refresh to change
  • We Use: W3.Hide(), W3.Show(), W3.ToggleShow()

Selectors Description Link
("*") Selects all elements in the document. Example
(this) Selects the current HTML element. Example
("p.intro") Selects all (P) elements with class="intro". Example
("div p") Selects all (P) element inside all (Div) elements. Example
("div p:first-child") Selects the first (P) element inside all (Div) elements. Example
("[href]") Selects all elements with an (href) attribute. Example
("a[target=_blank]") Selects all (a) elements with a target attribute value equal to "_blank". Example
("p:nth-child(even)") Selects all even (P) elements. Example
